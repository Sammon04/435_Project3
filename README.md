This is the 3rd project for my CIS 435 web design class at UofM Dearborn.
The goal of this project is to create a multi-page application to allow the user to create, list, filter, and delete tasks.
Each task has a title and may have a description, priority, and due date.
Tasks are stored in a JSON file on the server, which acts as a database.
The project makes heavy use of PHP for processing data, most making use of GET and POST requests, as well as PHP superglobals.
